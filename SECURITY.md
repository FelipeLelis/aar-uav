# Segurança

Se encontrar uma vulnerabilidade, dado sensível exposto ou comportamento que possa comprometer o projeto, evite abrir uma issue pública com detalhes exploráveis.

Use um canal privado com o mantenedor do repositório ou abra uma issue pública apenas com uma descrição geral, sem tokens, logs sensíveis ou dados pessoais.

## Escopo

- Código do simulador AAR.
- Módulo UAV / PyThrust integrado ao projeto.
- Configurações de deploy e automação.

## Fora de escopo

- Forks de terceiros sem relação direta com este repositório.
- Dependências externas não modificadas neste projeto.
